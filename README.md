# Account Manager - Java 8 Edition

A Java 8 implementation of an account management system, designed to be compatible with Slicer4J for dynamic program slicing analysis.

## Features

- Create and manage bank accounts
- Perform deposits, withdrawals, and transfers
- Track transaction history
- Account activation/deactivation
- Thread-safe operations ready for Slicer4J analysis

## Requirements

- Java 8 (JDK 1.8)
- Maven 3.x
- Slicer4J (for slicing analysis)
